HIV-1 Matrix builder Instruction

Welcome Massage

This site will provide you with the tools and guide to develop on HIV-1 Matrix using methods described by Weijie Sun.

Technical overview

3 main components are contained in script

  • calculated all HIV-1 MA trimer center coordinations

  • Generate PyMol Model from 1st component result

  • Generate tabulate relationship between trimer numbers and radius of HIV

System Requrements

Python 3 support (Recommend)

  • check prerequisites

Python 3.5 or greater

Anaconda3 > Jupytor NoteBook

Python Env Library in Jupytor NoteBook:

basemap 1.2.0 or greater

pymol 2.3.2 or greater

pandas 0.24.2 or greater

  • Getting Start

Open Anaconda create a python 3.5 environment

Install basemap, PyMol, panda libraries

If any libraries are not in your Anaconda packages, you could following the commands bellow

conda install -c anaconda basemap 
conda install -c anaconda pandas 
conda install -c schrodinger pymol

Then open the main_flow.ipynb file.

you should run the code successfully like the following results.

Python 2.7 support

1st component:

  • Check prerequisites

Python 2.7

Python Library:

importlib, matplotlib, mpl_toolkits

PyMol 1.72 or greater

  • Running of 1st component example
>python calculate_trimer_coordinates.py
Input the R: 
632
Do you want to use default r = 52.0769942809? (y or n): 
y

Then you will receive the following out message and two output files >> 'points' and 'lines'

There are 1024 points.

There are 1024 points' coordinations in 'points' file and There is another file 'lines' could provide the relationship between the center of trimers coordination

There is an option to quickly view what the lines and plot look like. You could input 3 to quit this script 1 - show and save the lines plot 2 - show and save the points model 3 - quit

2nd Component

  • Sample of running
python pymol_plot_trimer.py

If machine support for Pymol, then Pymol application will be open and running the script in pymol application.

You could store PyMol model in pdb or dae type of file.

3rd Component

  • Sample of running
> python tabulate_relationship_of_R_and_trimer_number.py

Then you need to input Min Radius, Max Radius, and interval

eg: Input the Min Radius: 600 Input the Max Radius: 700 Input the interval: 10 Do you want to use default r = 52.0769942809? (y or n): y

Then the script will output how many trimers be generated.

eg: 909 955 971 1019 1033 1074 1124 1157 1178 1185 1243
