Logistical Regression

Given x, want y^hat = P(y=1|x)

x \in R^{n_x}

Parameters: w \in R^{n_x}, b \in R

Output y^hat = W^T x + b , we except y^hat in [0, 1]

Since the W^T x + b could be negative or > 1. then it loss the meaning.

then we use the sigmoid function

y^hat = sigmoid(W^T x + b)

sigmoid(Z) = \frac{1}{1+e^{-z}}

another notation

x_0 = 1, x \in R^{n_x + 1}

y^hat = sigmoid(\theta^T x)

theta = [theta_0] <- b
[theta_1] <- w
[theta_2] <- w
….
[theta_n_x] <- w

Logistc Regression cost function

Given {(x^(1), y(1)), …, (x^(m), y^(m))}, want y^hat(i) = y^(i).

Loss(error) function: (for single one)

L(y^hat, y) = 1/2(y^hat - y)^2, It is a reasonable loss function, but Gradient decent could not work well.

L(y^hat, y) = - (y \log(y^hat) + (1-y) \log(1-y^hat))

if y = 1, L(y^hat, y) = -\log(y^hat), we want y^hat large

if y = 0, L(y^hat, y) = -\log(1-y^hat), we want log(1-y^hat) large
